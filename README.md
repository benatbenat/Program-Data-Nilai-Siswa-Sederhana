# Program-Data-Nilai-Siswa-Sederhana
Program untuk input data nilai siswa dengan penggunaan fungsi sederhana python (tanpa menggunakan dictionary)

## About The Project

<img width="337" height="220" alt="Modul 2 Capstone Project" src="https://github.com/user-attachments/assets/a999cb1e-2f33-4e96-940c-edb66dd014d6" />

Program ini dibuat untuk memenuhi syarat sebagai capstone project modul 2 sebagai pembelajaran fungsi-fungsi dari python.

### Built With

Visual Studio Code and Python

## Getting Started

### Prerequisites

Untuk menggunakan program ini, harap sudah menginstall aplikasi Visual Studio Code dan Python terlebih dahulu.
Format data siswa yang akan digunakan adalah [No (auto generated), NIS (auto generated), Nama, Kelas, Jurusan, Mata Pelajaran, Nilai, Status (Auto Generated, >= 75 = Lulus, selain itu akan terisi Remedial)]

### Installation

1. Download file.py terlebih dahulu
2. Letakkan file.py yang sudah di download di folder python anda
3. Locate folder python anda di dalam Visual Studio Code
4. Pilih file.py dari folder yang anda sudah pilih
5. Jalankan filenya
